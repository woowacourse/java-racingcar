## 📈 기능 목록

## 도메인

### 자동차

- [ ] 0에서 9 사이의 random 값에 따라 움직인다.
    - [ ] 0에서 9 사이에서 random 값을 구한다.
    - [ ] 4 이상인 경우 전진한다.
    - [ ] 3 이하인 경우 멈춘다.

- [x] [예외처리] 자동차의 이름은 5글자 이아만 가능하다.

## 게임

### 게임 준비

- [ ] 경주할 자동차 이름을 입력받는다.
    - [ ] [예외처리] 자동차가 1대 이하인 경우

- [ ] 시도할 횟수를 입력받는다.
    - [ ] [예외처리] 숫자가 아닌 경우

### 게임 진행

- [ ] lap만큼 반복한다.

### 결과 출력

- [ ] 게임을 끝났을 경우
    - [ ] 가장 많이 전진한 자동차를 출력한다.
