# java-racingcar
자동차 경주 게임 미션 저장소

## 👑 구현할 기능 목록
- __경주에 참여할 n대의 자동차 이름 입력 받기__
    - [조건] 자동차 이름은 쉼표로 구분
    - [조건] 자동차 이름은 5자 이하일 것

- __몇 번의 이동을 할 것인가 입력 받기__

- __주어진 횟수 동안 n대 자동차를 이동시키기__
    - [조건] 0~9 사이 랜덤 숫자를 선정, 4 이상 전진, 3 이하 멈춤
    - [조건] 매 이동 횟수 마다 자동차 이름과 거리를 출력한다.

- __게임 완료 후 우승자 출력하기__
    - [조건] 우승자는 한 명 이상일 수 있음
