# java-racingcar

자동차 경주 미션 저장소

## 🚀기능 목록

#### 도메인
- [ ] 자동차
  - [x] 전진 조건에 따라 전진할 수 있다
  - [x] 전진 조건을 판단할 수 있다.
    - [x] 0~9 의 값을 받아 4이상이면 전진 한다.
  - [x] 이름을 가지고 있다.
- [x] 모든 차를 n회 전진 또는 정지 한다.
  - [x] 모든 차를 전진 또는 정지 한다.
- [x] 우승한 자동차를 뽑는다. 

#### 뷰
- [x] 자동차의 이름을 입력받는다.
- [x] 시도 횟수를 입력받는다.
- [x] n회 시도한 결과를 출력한다.

## 예외사항
- [x] 자동차 이름은 5자를 초과하면 안된다.
- [x] 시도 횟수는 1회 이상이어야 한다.
