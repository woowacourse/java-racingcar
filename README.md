## 🚀 기능목록
* [x] 자동차 이름 입력하기
  * [x] 컴마로 이름을 구분
  * [x] 이름은 5자 이하
  * [x] 공백 허용안함
  * [x] 중복된 이름 허용안함
* [x] 자동차 객체를 관리하는 기능
  * [x] 자동차 객체를 저장
  * [x] 자동차 이름을 이용해 조회
  * [x] 저장된 전체 자동차 조회
* [x] 시도횟수 입력하기
  * [x] 자연수만 허용
* 난수 생성
  * 0 ~ 9 사이의 랜덤  생성
* [x] 자동차 이동
  * [x] 난수를 이용해 4이상일 경우 전진, 3이하일 경우 정지  
* [x] 한턴의 게임 진행
* [x] 우승자 찾기
  * [x] 가장 멀리 이동한 자동차가 우승
  * [x] 공동 우승 가능
